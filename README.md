
# Simple Expense Tracker
 a lightweight budget tracking app built to practice the fundamentals of React.


### What it does
- **Add Expenses:** input your spending with a title and amount.
- **Auto Calculate:** the app automatically updates the total sum as you add or remove items.


### Tech Stack
- **React.js:** handling the ui and state logic.
- **Tailwind CSS:** for the layout and styling.

### Setup

1. clone the project
```bash
git clone https://github.com/Aflah-Paruthicode/AF_ExpenseTracker.git
```

2. navigate into the project folder
```bash
cd ./AF_ExpenseTracker
```

3. install dependencies
```bash
npm install
```

4. start building
```bash
npm run start
```


### Why I built this
this was a key project during my early days of learning react. it helped me understand:
1. how to manage **lists and keys**.
2. how to use the `useState` hook to update the ui dynamically.
3. how to handle form inputs in a "react way."
